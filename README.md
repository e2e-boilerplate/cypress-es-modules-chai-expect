# Cypress Boilerplate
    
Cypress end-to-end test automation boilerplate, using ES Modules, Chai and Expect.
    
## Getting Started
    	1. git clone git@github.com:e2e-boilerplate/cypress-es-modules-chai-expect.git
    	2. cd cypress-es-modules-chai-expect
    	3. npm install
    	4. npm run test
